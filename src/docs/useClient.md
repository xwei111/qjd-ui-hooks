# UseClient

## 介绍

```useClient```可以获取浏览器可视区域尺寸

## 使用

```js

const { clientWidth, clientHeight } = useClient()
```

## 出参

| 参数        | 说明         | 类型        | 可选值        | 默认值  |
|-------------|--------------|-------------|--------------|---------|
| clientWidth | 可视区域宽度 | number | - | - |
| clientHeight | 可视区域高度 | number | - | - |
